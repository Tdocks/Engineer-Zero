# Applied AI Operations Engineer

This version is the original, interactive-first curriculum for the Applied AI Operations Engineer track. Its runtime manifest is defined in `lib/aio-content.ts`, where each module includes a stable ID, objectives, instruction sections, worked example, misconceptions, knowledge checks, artifact schema, deterministic completion rules, official-source references, and review state.

The browser receives only public instructional content. Answer keys remain in the protected grading path.

## Course paths

- **Interview Literacy Speedrun:** an independently accessible 48-hour core, plus five optional Days 3–7 extensions. It awards Know and Practice evidence, not independent implementation claims.
- **Zero-to-Role Program:** Foundation Bridge (weeks 1–6), Applied AI Fast Track (weeks 7–14), and Master/Capstone (weeks 15–18).
- **Role Concepts Library:** 36 source-backed Know cards. They teach vocabulary, operating relevance, red flags, and accountable handoffs without pretending the learner can configure every system.
- **Interview Studio:** a structured bank of independently authored technical, leadership, behavioral, and project-defense prompts.
